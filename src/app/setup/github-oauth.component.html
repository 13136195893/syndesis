Getting Started in Syndesis

Welcome to Syndesis! To get started you will need to register and authorize Syndesis on GitHub and select a location for storing integrations.

1

Register Syndesis

To start using Syndesis with GitHub, register Syndesis as an OAuth application in GitHub.

  Learn how to register an OAuth app on GitHub.
Success! Future changes can be made in Syndesis Global Settings / GitHub Settings.

2

GitHub Authorization

The Client ID and Client Secret generated in the first step will be used here to grant authorization for Syndesis to act on your behalf.

Learn how to get client ID and client secret on GitHub.

3

Configure Integration Location

Specify where to store the integration.
No account connected.

Success! You're ready to start in Syndesis.

Go to Syndesis home page to start using our awesome services! As always, you can go back and change the settings later on the GitHub Settings page.
