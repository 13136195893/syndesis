= Developing {prodname} extensions

integration, then an expert developer 
can code an extension that provides the needed behavior.  

The 
https://github.com/syndesisio/syndesis-extensions[Syndesis extensions repository]
contains examples of extensions. The following topics describe the 
requirements and provide additional examples:
